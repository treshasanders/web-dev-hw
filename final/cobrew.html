COBREW Website Redesign

COBREW, Marketing Technologist

Project Description

After doing freelance work with COBREW for a couple months, I was then hired as the company's Marketing Technologist. I have many responsibilities and one of them is wed design and updates. Since I first viewed the COBREW website, I'd wanted to get my hands on it and improve it and I was able to prove myself enought to be hired and asked to do such. Using my knowledge of user-interaction based design and web programming, I programmed the an update to the website. The intent of the website is to engage and inform brewers and draw acquire brewery interest in acquiring the COBREW System.

What I Learned

This was my first time being paid to do web design. I wasn't able to build the website from scratch, as the site had already been up for awhile. It was created using SquareSpace, so I had to start with the already created site and transform it into a site that I felt would bring in more traffic. This was my first time working with SquareSpace. In my designer opinion, I think my update is a major improvement from the previous version. There will be another update to the webstie in 2018 that should make the site even better. Not only will the site be for brewers to learn more but also for brewery customers who care about sustainability. We want to create a community of craft beer and sustainability enthusiasts and spread the word about COBREW from every perspective. As I continue to work with COBREW and run analytics on the site and social media, I'm making changes that optimize the user experience. I'm constantly learning and growing in my job.
